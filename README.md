# Fusion MCP Projekt

## Voraussetzungen
- Autodesk Fusion 360
- Claude Desktop
- Python 3.11+
- MCP-CLI (`uv`)
  

## Installation

1. Pakete installieren:
```bash

gitclone
cd Server
pip install -r requirements.txt
uv run mcp install MCP_Server.py 
```
# Fusion Addin einbetten
```bash
python install_addin.py
```
